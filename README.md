# Student Performance Predictor

This project uses a simple machine learning model to predict whether a student will pass or fail based on study hours.

## Technologies Used
- Python
- NumPy
- Scikit-learn

## Concepts Used
- Machine Learning
- Logistic Regression
- Model Training and Prediction

## How to Run
1. Run the Python script
2. Enter number of study hours
3. The program predicts pass or fail

## Output Example
Enter study hours: 5  
Prediction: Student will PASS
